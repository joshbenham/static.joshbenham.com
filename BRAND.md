# BRAND.md

## Source of truth
This project follows the current Josh Benham and HamForge landing page style.

## Direction
- flat UI
- dark slate background
- orange accent
- white headings
- muted slate body text
- rounded cards
- subtle hover states
- no gradients
- lighter strokes on boxes

## Typography
- Inter
- fallback: Arial, Helvetica, sans-serif

## Josh Benham page rules
- keep the page simple and portfolio-like
- preserve current content and structure unless explicitly asked
- keep the same dark/orange/white design system
- use flat buttons and flat panels
