# Josh Benham Tailwind Project

This is a static Tailwind landing page project converted from the provided `joshbenham-updated.html`.

## Included
- `public/index.html`
- `src/input.css`
- `tailwind.config.js`
- `postcss.config.js`
- `package.json`
- `README.md`
- `DEPLOY.md`
- `BRAND.md`
- `public/images/README.txt`

## Run locally

1. Install dependencies

```bash
npm install
```

2. Build the CSS

```bash
npm run build
```

3. Watch while editing

```bash
npm run watch
```

4. Open `public/index.html` in your browser after building.

## Notes
- Content and structure were kept the same.
- The Tailwind CDN script was replaced with `./assets/app.css`.
- Add your real logos/icons into `public/images/` when ready.
